直接调用是调用 NTSilk 最简单的方法。

NTSilk Userland 提供了 NTSilk 的预编译版本，这是一个包含了 NTSilk、wav、MP3 等众多编码器的 FFmpeg CLI。可以通过 CLI 直接完成 SILK 与其他格式的相互转换。

## 安装

可以从本站直接下载 NTSilk Userland 提供的 NTSilk：

- **Windows x64**： https://ntsilk.ilharper.com/ntsilk-win32-x64.exe
- **Linux x64**： https://ntsilk.ilharper.com/ntsilk-linux-x64

若希望从 GitHub 上下载，或需要 NTSilk 的历史版本，可以前往 [NTSilk Binaries](https://github.com/ntsilk-userland/binaries)。

## CLI

可以在终端中直接使用上面下载的 CLI。

## Python

例如，将 `my.mp3` 转码为 `my.ntsilk`：

```py
import subprocess

subprocess.run(["D:/ntsilk.exe", "-i", "my.mp3", "my.ntsilk"])
```

[`subprocess` 模块](https://docs.python.org/zh-cn/3/library/subprocess.html)
的文档提供了详细的用法说明。

## Node.js

例如，将 `my.mp3` 转码为 `my.ntsilk`：

```ts
const { execFileSync } = require('node:child_process')

execFileSync('D:/ntsilk.exe', ['-i', 'my.mp3', 'my.ntsilk'])
```

[`child_process` 模块](https://nodejs.org/docs/latest/api/child_process.html)
的文档提供了详细的用法说明。
