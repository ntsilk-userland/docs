欢迎使用 NTSilk。

NTSilk 为 FFmpeg 提供了 SILK 编码和解码支持，使你能够使用 FFmpeg 在一步内完成
wav、MP3 等格式音频到 SILK 音频的互转。

可以直接使用集成了 NTSilk 的框架，也可以直接调用 NTSilk。向下翻动查看所有使用方式。

## 集成

一些框架已经提供了 NTSilk 的集成支持。使用这些框架时，框架会自动调用 NTSilk
完成操作，你可以直接发送或接收 MP3 格式的音频文件。
